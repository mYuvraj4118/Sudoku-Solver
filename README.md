# 🧩 Sudoku Solver

A simple and interactive **Sudoku Solver Web Application** built using **HTML, CSS, and JavaScript**.  
This project allows users to solve Sudoku puzzles directly in the browser with an easy-to-use interface.

---

## 🚀 Features

- ✅ Interactive 9×9 Sudoku grid
- ✅ User-friendly interface
- ✅ Automatically solves Sudoku puzzles
- ✅ Validates Sudoku constraints
- ✅ Responsive design
- ✅ Pure frontend implementation (No backend required)

---

## 🛠️ Technologies Used

| Technology | Purpose                                |
| ---------- | -------------------------------------- |
| HTML5      | Structure of the application           |
| CSS3       | Styling and responsive design          |
| JavaScript | Sudoku solving logic and functionality |

---

## 🧠 How It Works

The Sudoku Solver uses a **Backtracking Algorithm** implemented in JavaScript to solve Sudoku puzzles.

The algorithm:

1. Finds an empty cell in the Sudoku grid.
2. Tries possible numbers from 1 to 9.
3. Checks whether the number is valid according to Sudoku rules.
4. Places the number if it is valid.
5. Recursively continues until the puzzle is solved.
6. Backtracks whenever an incorrect choice is detected.


### 2. Open the project folder

```bash
cd Sudoku-Solver
```

### 3. Run the application

Open the `index.html` file in your browser.

## 🎮 How to Use

1. Open the Sudoku Solver application.
2. Enter numbers into the Sudoku grid.
3. Keep empty cells blank.
4. Click the **Solve** button.
5. The application will automatically generate the solution.
